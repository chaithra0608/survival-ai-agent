# Survival AI Agent using Q-Learning

## Overview

This project demonstrates a simple Reinforcement Learning (RL) environment where an AI agent learns to survive using the Q-learning algorithm. The agent explores a grid world, receives rewards for reaching the food, penalties for encountering the demon, and gradually learns the optimal path through trial and error.

The project showcases the core concepts of reinforcement learning, including state representation, action selection, reward-based learning, and Q-table updates.

## Features

* Grid-based environment
* Q-Learning algorithm
* Reward and penalty system
* Agent learns optimal actions over multiple episodes
* Exploration vs. exploitation using epsilon-greedy policy

## Technologies Used

* Python 3
* Q-Learning
* Reinforcement Learning Concepts
* Standard Python Libraries
